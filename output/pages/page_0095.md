Gastos financieros
menos
Productos financieros
igual
Gastos de venta más Gastos de administración más Pérdida financiera igual Gastos de operación

Ejemplo:
Supongamos los mismos gastos de venta, de administración y financieros del caso anterior, y además, los siguientes productos: intereses cobrados $ 1 000.00 y utilidad en cambios $ 3 000.00.

| Concepto | (1) | (2) | (3) | (4) |
| :--- | :--- | :--- | :--- | :--- |
| **Gastos de operación** | | | | |
| **Gastos de venta** | | | | |
| Renta del almacén | | $ 17 000 00 | | |
| Propaganda | | 9 000 00 | | |
| Sueldos de agentes y dependientes | | 32 000 00 | | |
| Comisiones de agentes | | 16 000 00 | | |
| Consumo de luz del almacén | | 1 000 00 | $ 75 000 00 | |
| **Gastos de administración** | | | | |
| Renta de las oficinas | | $ 12 000 00 | | |
| Sueldos del personal de oficinas | | 43 000 00 | | |
| Papelería y útiles de oficina | | 3 000 00 | | |
| Consumo de luz de oficinas | | 2 000 00 | 60 000 00 | |
| **Gastos financieros** | | | | |
| Intereses pagados | $ 5 000 00 | | | |
| Pérdida en cambios | 4 500 00 | | | |
| Gastos de situación | 500 00 | $ 10 000 00 | | |
| **Productos financieros** | | | | |
| Intereses cobrados | $ 1 000 00 | | | |
| Utilidad en cambios | 3 000 00 | 4 000 00 | 6 000 00 | $ 141 000 00 |

Como puede verse, primero se clasificaron los gastos financieros, por ser mayor su valor, y después se clasificaron los productos financieros, la diferencia entre ambos fue la pérdida financiera, la cual se sumó a los gastos de venta más los de administración, para obtener de esta manera el total de los gastos de operación.

Cuando el resultado obtenido sea utilidad financiera, su valor se debe restar de la suma de los gastos de venta con los de administración, para obtener el total de los gastos de operación.
