**Saldo deudor.** Una cuenta tiene saldo deudor cuando su movimiento deudor es mayor que el acreedor. Ejemplo:

| **Debe** | **Cuenta: NOMBRE DE LA CUENTA** | **Haber** |
|---|---|---|
| 120 000.00 | | 110 000.00 |
| 60 000.00 | | 30 000.00 |
| 20 000.00 | | 10 000.00 |
| **200 000.00** | *Movimiento deudor* | **150 000.00** |
| **50 000.00** | *Saldo deudor* | |

**Saldo acreedor.** Una cuenta tiene saldo acreedor cuando su movimiento acreedor es mayor que el deudor. Ejemplo:

| **Debe** | **Cuenta: NOMBRE DE LA CUENTA** | **Haber** |
|---|---|---|
| 110 000.00 | | 130 000.00 |
| 30 000.00 | | 20 000.00 |
| 10 000.00 | | 30 000.00 |
| **150 000.00** | *Movimiento deudor* | **180 000.00** |
| | *Saldo acreedor* | **30 000.00** |

Cuando una cuenta únicamente tiene movimiento deudor, su valor representa saldo deudor. Ejemplo:

| **Debe** | **Cuenta: NOMBRE DE LA CUENTA** | **Haber** |
|---|---|---|
| 100 000.00 | | |
| 30 000.00 | | |
| 50 000.00 | | |
| **180 000.00** | *Movimiento deudor* | |
| **180 000.00** | *Saldo deudor* | |

En caso contrario, el saldo será acreedor.

**Cuenta saldada o cerrada.** Una cuenta está saldada o cerrada cuando sus movimientos son iguales. Ejemplo:

| **Debe** | **Cuenta: NOMBRE DE LA CUENTA** | **Haber** |
|---|---|---|
| 170 000.00 | | 140 000.00 |
| 80 000.00 | | 60 000.00 |
| 50 000.00 | | 100 000.00 |
| **300 000.00** | *Movimiento deudor* | **300 000.00** |
| | *Saldada* | |
