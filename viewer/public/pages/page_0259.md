1. Por una venta de mercancías que importó $ 30 000.00 en efectivo, se corrió el siguiente asiento:

| | | | |
|---|---|---|---|
| | Ventas | $ 30 000 | 00 |
| | IVA por pagar | 4 500 | 00 |
| | Caja | $ 34 500 | 00 |

a) El error se advirtió antes de hacer el pase al mayor.
b) El error se advirtió después de haber hecho el pase al mayor.
