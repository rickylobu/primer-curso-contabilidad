| **Debe** | **Cuenta: EQUIPO DE REPARTO** | **Haber** |
|---|---|---|
| **Al principiar el ejercicio** | | **Durante el ejercicio** |

| **Durante el ejercicio** | | **Al terminar el ejercicio** |
| 2. Del precio de costo de los vehículos que se adquieran. | | 2. Del precio de costo de los vehículos que se den de baja por inservibles. |
| | | 3. Del valor de su saldo, para saldarla. |
