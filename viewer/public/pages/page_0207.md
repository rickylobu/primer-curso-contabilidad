Para saber el importe a pagar de impuesto al valor agregado de la declaración mensual es preciso determinar lo siguiente:

a) El factor del impuesto acreditable.
b) El impuesto acreditable.
c) El impuesto trasladado.

### Factor del impuesto acreditable. 
El factor del impuesto acreditable se determina dividiendo las ventas con gravamen entre el total de ventas. Ejemplo:

| | |
| :--- | :--- |
| Ventas con gravamen | $ 1 700 000.00 |
| Ventas sin gravamen | 900 000.00 |
| Total de ventas | $ 2 600 000.00 |

`Factor del impuesto acreditable = 1 700 000.00 / 2 600 000.00 = 0.65 = 65%`

### Impuesto acreditable. 
El valor del impuesto acreditable se obtiene multiplicando el IVA por acreditar por el factor del impuesto acreditable. Ejemplo:

| Operaciones | IVA por acreditar | | Factor | = | Impuesto acreditable |
| :--- | :--- | :--- | :--- | :--- | :--- |
| Compras de mercancías | $ 210 000.00 | X | 0.65 | = | $ 136 500.00 |
| Gastos de venta | 45 000.00 | X | 0.65 | = | 29 250.00 |
| Gastos de administración | 22 500.00 | X | 0.65 | = | 14 625.00 |
| **Valor del impuesto acreditable** | | | | | **$ 180 375.00** |

### Impuesto trasladado. 
El valor del impuesto trasladado a los clientes se determina aplicando a las ventas de mercancías las tasas correspondientes.

| Operaciones | Valor | Tasas | Impuesto trasladado |
| :--- | :--- | :--- | :--- |
| Ventas de mercancías con gravamen | $ 1 700 000.00 | 15% | $ 255 000.00 |
| Ventas de mercancías sin gravamen | 900 000.00 | 0% | 0.00 |
| **Valor del impuesto trasladado** | | | **$ 255 000.00** |

### Impuesto a pagar. 
El valor del impuesto a pagar, en la declaración de pago provisional, se obtiene restando del impuesto trasladado el impuesto acreditable. Ejemplo:

| | |
| :--- | :--- |
| Impuesto trasladado | $ 255 000.00 |
| Impuesto acreditable | 180 375.00 |
| **Impuesto a pagar** | **$ 74 625.00** |
